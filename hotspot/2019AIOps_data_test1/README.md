# 2019AIOpsdata_test1
#### 数据描述
1. 数据集有连续14天的数据，时间间隔为5分钟，每个文件名以时间戳（精度为毫秒）命名，对应数据相应的时间点。

2. 每个文件为“.csv”格式文件，其中数据以“,”间隔。每个文件中的有6列数据，前5列为五个维度的属性值，第6列是前5列属性值组合对应的数值。  

3. 此数据集为实际中采集的原始数据，但进行了注入异常的操作。对应的人工注入异常的时间戳为Anomalytime_data_test1.csv中的时间戳。
